# Flow Race：循环跑图

## 流程图

```mermaid
flowchart TB
    A["开始 Flow Race"] --> B["进入我的车辆"]
    B --> C{"当前车辆是<br/>R917 SkillCar？"}
    C -- "否" --> D["识别并切换 R917"]
    C -- "是" --> E["保留车辆列表状态"]
    D --> E

    E --> F["进入主菜单与 EventLab"]
    F --> G["识别并加入挑战"]
    G --> H["等待自动发车"]
    H --> I["持续按住 W + Up"]

    I --> J{"识别比赛结果"}
    J -- "挑战完成" --> K["完成本轮计数"]
    J -- "挑战失败" --> L["失败同样计数"]
    J -- "超时" --> M["尝试重开或交给全局恢复"]

    K --> N{"最后一轮？"}
    L --> N
    N -- "否" --> O["进入下一轮挑战"]
    N -- "是" --> P["退出挑战"]

    P --> Q{"出现挑战评分？"}
    Q -- "是" --> R["Down → Enter<br/>执行点踩"]
    Q -- "否" --> S["结束 Flow Race"]
    R --> S
```

## 关键实现

- 使用 `skillcar_r917.png` 确认或切换刷图车辆。
- 通过模板匹配定位 EventLab、加入挑战和比赛结果。
- 比赛期间持续保持 `W + Up`。
- 挑战完成和挑战失败都会计入本轮次数。
- 末轮失败时按 Esc 退出，不再重试。
- 挑战评分弹窗为可选页面，出现时执行点踩。
- 超时或页面状态异常时交给统一恢复逻辑。
